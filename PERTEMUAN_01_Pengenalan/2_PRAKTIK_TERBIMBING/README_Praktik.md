# ✍️ Praktik Terbimbing: Navigasi & UI Blender

## File Latihan
* `praktik_01_navigasi.blend`

File ini berisi tiga objek primitif dasar (Cube, Sphere, Cone) dan sebuah Plane sebagai lantai.

## Instruksi Praktik
Sesi ini dipandu oleh dosen/tutorial untuk membiasakan diri dengan:

1.  **Antarmuka (UI):**
    * **3D Viewport:** Jendela utama tempat kita bekerja.
    * **Outliner:** Daftar semua objek dalam *scene* (di kanan atas).
    * **Properties Panel:** Panel berisi semua pengaturan (di kanan bawah).
    * **Timeline:** Panel animasi (di bawah).

2.  **Navigasi Viewport (WAJIB HAFAL):**
    * **Orbit (Memutar):** Tekan dan tahan **Tombol Tengah Mouse (MMB)**.
    * **Pan (Menggeser):** Tahan **Shift + MMB**.
    * **Zoom (Memperbesar/kecil):** **Scroll Wheel** atau **Ctrl + MMB**.
    * **Fokus ke Objek:** Pilih objek (klik kiri), lalu tekan **Numpad . (titik)**.
    * **Lihat Semua Objek:** Tekan **Tombol Home**.
