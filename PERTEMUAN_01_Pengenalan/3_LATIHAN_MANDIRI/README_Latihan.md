## Tugas
Membuat sebuah *scene* sederhana menggunakan *default startup file* Blender, hanya dengan menggunakan manipulasi objek dasar.

## Instruksi
1.  Buka Blender (gunakan *startup file* awal yang berisi 1 Cube, 1 Camera, 1 Light).
2.  **Jangan hapus Cube** tersebut.
3.  Tambahkan dua objek baru: **Sphere** dan **Cone** (Gunakan `Shift + A` > `Mesh` > `UV Sphere` dan `Shift + A` > `Mesh` > `Cone`).
4.  Tambahkan satu objek **Plane** (`Shift + A` > `Mesh` > `Plane`) sebagai lantai.
5.  Gunakan hanya tiga alat transformasi dasar:
    * **Grab (G):** Untuk memindahkan objek (Gunakan sumbu `X`, `Y`, `Z` untuk presisi, misal `G` lalu `X`).
    * **Rotate (R):** Untuk memutar objek.
    * **Scale (S):** Untuk mengubah ukuran objek.
6.  **Tantangan:** Atur ketiga objek (Cube, Sphere, Cone) di atas Plane untuk menciptakan sebuah komposisi yang menarik secara visual. Posisikan mereka agar tidak saling menumpuk.
7.  Atur posisi Kamera (`0` di Numpad untuk melihat dari kamera) agar komposisi Anda terlihat bagus.
8.  Render gambar Anda (`F12`).
9.  Simpan hasil render sebagai `latihan_01_scena_pertama.png`.
10. Simpan file kerja Anda sebagai `latihan_01_scena_pertama.blend`.

## File
* `latihan_01_scena_pertama.blend` (File kerja Blender)
* `latihan_01_scena_pertama.png` (Hasil render final)
